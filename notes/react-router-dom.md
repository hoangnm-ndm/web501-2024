# 1: Cai dat:

```
npm install react-router-dom
```

# 2: Su dung:

- Them BrowserRouter vao main.jsx

```js
import { BrowserRouter } from "react-router-dom";
// ...
<BrowserRouter>
	<App />
</BrowserRouter>;
```

- Cấu hình lại App.jsx
- Vào Header.jsx đổi thẻ "a" thành thẻ "Link", đổi "href" thành "to"
