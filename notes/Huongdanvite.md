# 1: Khởi tạo dự án:

```
npm create vite
```

Có một số tùy chọn:

1. Đặt tên dự án (Nếu chọn thư mục hiện tại là thư mục gốc thì ấn dấu ".")
2. Chọn framework (React)
3. Chọn variant (Javascript + SWC)

# 2. Khởi chạy

- Đứng tại thư mục gốc của dự án và chạy câu lệnh:

```
npm install
npm run dev
```
